mir-commons

Utilities for Java programming with Spring and Hibernate.

  • mir-commons-base-model module provides a DAO layer implementation right out-of-the-box, using Java Generics.

  • mir-commons-base-utils provides common utilities to handle standard Java objects like String, Date, File, Collection, ecc..

  • mir-commons-db-utils provides features to access DB without Hibernate, ideal for simple projects or feasibility tests.

  • mir-commons-web-utils provides features to easily use Spring MVC, both with UI and REST API.
